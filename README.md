# FSM Password Locker

## Overview

A Verilog-based finite state machine (FSM) password locker.

Password sequence: 1101

Features:
- Serial password entry
- Submit button verification
- Failed attempt counter
- Permanent lock after 3 incorrect attempts
- Reset current entry
- System reset

## Files

- locker.v : RTL Design
- tb.v : Testbench
- waveform.png : Simulation result

## Tools Used

- Verilog HDL
- Vivado Simulator
